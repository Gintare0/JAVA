# Lucky Ticket

Java programa, kuri nustato, ar loterijos bilietas yra laimingas.

## Užduoties sąlygos

Loterijos bilietas laikomas laimingu, jei:
Jis sudarytas iš 6 skaitmenų
Visi skaitmenys yra skirtingi
Pirmų trijų skaitmenų suma lygi paskutinių trijų skaitmenų sumai

Programa:
1. Patikrina bilietą, perduotą per komandinę eilutę
2. Generuoja atsitiktinius bilietus, kol randa pirmą laimingą
3. Išveda, kiek bandymų reikėjo

## Paleidimas

```bash
javac LuckyTicket.java
java LuckyTicket 628745
